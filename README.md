# housescraper

### Commands

Run

```
node index.js
```

Debug

```
node --inspect index.js
```

## Deployment

Deployed to Heroku: push to master branch on Github.

Check app logs in Heroku

```
heroku logs
```

## Built With

* [Firestore](https://firebase.google.com/docs/firestore) - Database
* [Heroku](https://www.heroku.com/) - Cloud Server
